# Software Factory Template

Bundled project-scoped scaffold for `lex factory init`.

Files under `.omp/extensions/software-factory/` provide runtime behavior.
Files under `.omp/factory/` provide repo-local config, prompts, workflows, memory policy, and verification scripts.
Files under `.omp/agents`, `.omp/prompts`, `.omp/rules`, and `.omp/skills` surface discoverable resources through lex native loaders.
