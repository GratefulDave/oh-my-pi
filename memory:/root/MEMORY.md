- **Opencode-Antigravity Config Global Isolation:** Project-level extension settings (`.omp/settings.json`) are only loaded in their own repositories. Global user configuration (`~/.omp/agent/config.yml`) did not auto-discover the antigravity-adapter relative paths, causing the opencode-antigravity provider to be missing in other directories. Resolved by adding the absolute path to the global `extensions:` list in `~/.omp/agent/config.yml`.
